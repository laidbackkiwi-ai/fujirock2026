# Fuji Rock Companion 2026 v7

## 新機能：グループモード
- 6桁のグループコードを作成
- 招待リンクをLINEなどで共有
- 仲間がリンクを開くとグループ名とコードを端末へ保存
- 現在地、行き先、集合までの時間、ひとことを共有
- GPS座標を任意でチェックインに添付
- 仲間から届いたチェックインリンクを開くとメンバーボードへ反映
- データは各端末のlocalStorageへ保存

## 重要：Version 7の共有方式
GitHub Pagesは静的サイトのため、Version 7はサーバーを使わない「リンク共有方式」です。
メンバーの位置や状況は自動同期されません。各メンバーが「状況を共有」を押し、
LINE等で届いたリンクを他のメンバーが開くことで更新されます。

リアルタイム自動同期・自動位置共有を実現するには、Firebase / Supabase等の
バックエンド設定が別途必要です。

## 更新方法
ZIPを展開し、GitHubリポジトリの全ファイルを上書きして Commit changes を押してください。
GitHub Pagesのデプロイ完了後、Safariで公開URLを開き直してください。
