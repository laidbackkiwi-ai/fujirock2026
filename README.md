# FUJI ROCK 2026 PWA

## 公開方法（最短）

### Netlify Drop
1. このフォルダ内のファイル一式を展開します。
2. Netlify Drop にフォルダをドラッグ＆ドロップします。
3. 発行された `https://...netlify.app` のURLをNotionに貼ります。

### GitHub Pages
1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルをすべてアップロードします。
3. Settings → Pages → Deploy from a branch を選択します。
4. `main` / `(root)` を選択して保存します。
5. 数分後に表示される公開URLをNotionに貼ります。

## iPhoneホーム画面への追加
Safariで公開URLを開き、共有 →「ホーム画面に追加」。

## 内容
- 5ステージ表
- 上段ステージ名固定
- 左端時間固定
- 時間順表示との切替
- ステージ絞り込み
- アーティスト公式紹介ページへのリンク
- オフラインキャッシュ
