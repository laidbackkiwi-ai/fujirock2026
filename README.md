# Fuji Rock Companion 2026 v5

## Version 5の追加機能
- 今日のコンシェルジュ
- 現在公演中／次のライブ／移動開始目安
- 通知許可時の移動開始通知（アプリを開いている間）
- GPSから最寄りステージを推定
- 苗場の現在天気と3時間降水確率（Open-Meteo）
- GPS移動距離と推定歩数のトラッカー
- 飲酒時刻・杯数メモと帰路の安全注意
- Version 4のYahoo!カーナビ／Googleマップ選択を継承

## 注意
- GPSのステージ判定は概算座標を使った参考表示です。
- iPhoneのWebアプリからヘルスケアの実歩数は取得できないため、歩数はGPS移動距離からの推定です。
- 通知はOS・ブラウザの制約上、アプリが閉じている間は保証されません。
- 天気取得には通信が必要です。最後に取得した情報は端末内に保持します。
