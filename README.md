# Fuji Rock Companion 2026 v7.1

## マイナーチェンジ
- Fujii KazeとKHRUANGBINの固定選択を解除
- 2組とも他の出演者と同じ「＋ 今日見る」ボタンで追加・解除可能
- 次のライブ、Today Concierge、表形式の強調表示には各端末で選択した出演者だけを使用
- 選択内容は同行者それぞれの端末に保存

Version 7のリンク共有型グループ機能を継続しています。

## 更新方法
ZIPを展開し、GitHubリポジトリ内のファイルをすべて上書きしてコミットしてください。
GitHub Pagesのデプロイ後、Safariで公開URLを再読み込みしてください。
古い表示が残る場合は公開URLの末尾に `?v=7.1` を付けて開いてください。
