# Fuji Rock Companion 2026 v6

## Arrival Planner
- 会場到着目標から市川・中野坂上・神立の時刻を逆算
- 土曜日早朝の「軽め／標準／混雑」3シナリオ
- 中野坂上での合流時間を変更可能
- 関越道での休憩をオン／オフ可能
- 当日の事故・渋滞による追加時間を0〜90分で補正
- 設定と計算結果は端末内に保存
- NEXCO渋滞予測、JARTIC、Googleマップの実時間確認ボタン

## 標準モデルの初期値
- 市川→中野坂上：50分
- 中野坂上→神立：170分
- 土曜日朝の混雑余裕：20分
- 休憩：15分
- 神立到着後：駐車10分＋待ち10分＋シャトル30分＋入場10分

このモデルは公式サイトからリアルタイム所要時間を自動取得するものではありません。
出発当日はNEXCO/JARTIC/Googleマップを確認し、見込まれる遅れを「当日の追加時間」に入力してください。
